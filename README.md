# Simple Interest Calculator
This program calculates the total value and total interest earned for a given principal, interest rate, and number of years.

## Usage
To use the program, run the SicRunner class. The program will prompt you to enter the principal, interest rate, and number of years. After entering these values, the program will calculate and output the total value and total interest earned.
